# InteropCustomApp

## Installation

run `npm install` pour installer toutes les dépendances nécéssaire au fonctionnement de l'application

## Setup

Créer un ficher .env avec dedans `SECRET_KEY=` puis mettez votre secret key unique

## Run Project 

Se rendre dans le répertoire ica-back puis run `node index.js`

## Admin et non Admin

Les admin n'auront pas enormemement plus d'accès que les non-admin, seulement la page http://localhost/custom-interop-team qui leur permettront de modifier le temps de livraion et plus a l'avenir
